# 第三方 API 数字字号调整

## 目标

仅缩小第三方 API 余额/额度数字，保留官方额度字号以及全部现有布局、颜色、阴影、动画和交互。

## 设计

- 官方额度：悬浮球 `26`，详情卡 `40`。
- 第三方余额：悬浮球 `18`，详情卡 `28`。
- `OrbViewModel` 根据当前展示的是官方额度还是第三方余额提供两处字号。
- 两个 XAML 文本节点绑定对应字号；不修改任何 Storyboard、Trigger 或动画设置。

## 验证

- ViewModel 测试覆盖官方额度与第三方余额两种字号。
- Windows 测试覆盖两处 XAML 的实际字号绑定。
- 运行现有 Core 与 Windows 测试，确认无回归。
