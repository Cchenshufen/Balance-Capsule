# Balance Capsule v1.2.15.14

本次修订提高 Windows 启动可靠性，并调整双平台最低系统要求。

## 下载

- macOS 26+ Apple Silicon：`BalanceCapsule-1.2.15-mac.14-arm64.dmg`
- Windows 11 x64：`BalanceCapsule-1.2.15-win.14-x64.zip`（推荐）
- Windows 11 x64 单文件：`BalanceCapsule-1.2.15-win.14-x64.exe`

## Windows 修复

- 仅支持 Windows 11 build 22000 或更新版本与 x64 处理器。
- 便携 ZIP 改为完整多文件自包含布局，不依赖单文件自解压。
- 单文件 EXE 关闭程序集压缩，降低首次启动时的解压兼容风险。
- 启动早期异常不再静默退出；现在显示错误，并写入 `%LOCALAPPDATA%\BalanceCapsule\startup.log`。
- CC Switch SQLite 变为按需初始化；未安装 CC Switch 或系统 SQLite 不可用时，不再影响主程序打开。

## macOS 调整

- 最低支持 macOS 26。
- 仅支持 Apple Silicon arm64，不支持 Intel Mac。

两个平台当前均未使用商业代码签名。请只从本仓库 Release 下载并校验 SHA-256。
