# Quota Orb v1.2.0

## 新增

- 托盘和悬浮球右键菜单支持 Codex / Claude 数据来源切换，并记住上次选择。
- 支持 Claude Code 2.1.80+ 官方 statusLine `rate_limits`，显示 5 小时和 7 天剩余额度。
- 支持只读识别 CC Switch 的当前 Claude 供应商。
- Claude Code 使用明确的 DeepSeek 或 OpenRouter HTTPS 上游时，显示对应剩余金额。

## 安全与边界

- Claude statusLine 桥接只缓存使用百分比、重置时间和采集时间。
- 不读取或保存 Claude OAuth 凭据、会话正文、工作目录、模型名和 session ID。
- 已存在自定义 statusLine 时不覆盖；未知、聚合或本地代理无法绑定上游时显示暂不支持。
- 第三方余额继续只允许 HTTPS、GET、同主机且禁止重定向。

## 兼容性

- Claude 官方额度要求 Claude Code 2.1.80 或更高版本，并在至少一次模型响应后同步。
- Claude Desktop 不执行 CLI `statusLine`；桌面版请使用模型旁的用量环，Quota Orb 自动同步仅支持终端版 Claude Code CLI。
- Claude Desktop 与 Claude Code 共用订阅限额，但单独使用桌面端不会触发本机 statusLine 更新。
- 首批 Claude 第三方余额适配器为 DeepSeek 和 OpenRouter。
