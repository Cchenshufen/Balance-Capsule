# Quota Orb v1.0.0

## 亮点

- 双层玻璃悬浮球、分层液位、刻度、微气泡和环境阴影。
- 安全、提醒、临界和读取失败四种清晰状态。
- 悬停展开玻璃控制台，查看可用窗口、重置时间和最近刷新时间。
- 支持拖动吸附、系统托盘、开机启动、全屏自动隐藏和减少动画。
- 支持官方 Codex 桌面版运行目录，不要求桌面版额外注册到 `PATH`。
- 每 15 秒自动读取本机官方 Codex 配额，不保存账号或额度快照。

## 安装

1. 下载 `QuotaOrb-win-x64.zip` 和 `SHA256SUMS.txt`。
2. 校验 ZIP 的 SHA-256。
3. 将 ZIP 解压到固定目录。
4. 运行 `QuotaOrb.Windows.exe`。

发布包是 Windows x64 自包含便携程序，不需要单独安装 .NET，也不需要管理员权限。

## 系统要求

- Windows 10 2004（build 19041）或更新版本，或 Windows 11
- x64 处理器
- 已使用 ChatGPT 账号登录的官方 Codex 桌面版，或已登录并位于 `PATH` 的官方 Codex CLI

## 已知边界

- 当前是便携 ZIP，不是安装器，也不包含自动更新。
- 程序尚未进行商业代码签名，Windows SmartScreen 可能在首次运行时提示未知发布者。
- 官方额度服务短暂波动时，当前刷新周期可能显示读取失败；下一次自动或手动刷新成功后恢复。
- 当前没有 macOS、Linux 或 Windows ARM 原生版本。
- Quota Orb 是独立项目，与 OpenAI 无隶属或背书关系。

## 校验

Release 同时提供 `SHA256SUMS.txt`。在 PowerShell 中运行：

```powershell
Get-FileHash .\QuotaOrb-win-x64.zip -Algorithm SHA256
Get-Content .\SHA256SUMS.txt
```

两个 SHA-256 值应完全一致。
