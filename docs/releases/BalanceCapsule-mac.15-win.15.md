# Balance Capsule mac.15 / win.15

Balance Capsule 现在使用按平台区分的简洁版本名称：

- macOS：`BalanceCapsule-mac.15`
- Windows：`BalanceCapsule-win.15`

## 下载文件

- macOS Apple Silicon：`BalanceCapsule-mac.15-arm64.dmg`
- macOS Apple Silicon 备用包：`BalanceCapsule-mac.15-arm64.zip`
- Windows 11 x64：`BalanceCapsule-win.15-x64.exe`
- Windows 11 x64 备用包：`BalanceCapsule-win.15-x64.zip`

macOS 仅支持 macOS 26 或更高版本的 Apple Silicon 设备；Windows 仅支持 Windows 11 x64。
