# Balance Capsule v1.2.15

Balance Capsule 是一款常驻桌面的 AI Agent 配额悬浮球。本次同时发布 macOS Apple Silicon `1.2.15-mac.13` 与 Windows x64 `1.2.15-win.13`。

## 下载

### macOS Apple Silicon

- `BalanceCapsule-1.2.15-mac.13-arm64.dmg`：推荐安装包。
- `BalanceCapsule-1.2.15-mac.13-arm64.zip`：ZIP 备用包。
- 要求 macOS 13 或更高版本，仅支持 M1/M2/M3/M4 及后续 Apple Silicon，不支持 Intel Mac。

### Windows x64

- `BalanceCapsule-1.2.15-win.13-x64.exe`：自包含单文件版本。
- `BalanceCapsule-1.2.15-win.13-x64.zip`：便携 ZIP。
- 要求 Windows 10 2004（build 19041）或更高版本，或 Windows 11，仅支持 x64。

## 核心功能

- 78px 液态玻璃悬浮球，可拖动并自动吸附屏幕边缘。
- 鼠标滑入自动展开详情面板，球体与面板之间呈连续液滴过渡。
- 双击悬浮球立即刷新；右键与托盘菜单支持刷新、切换来源、显示或隐藏悬浮球。
- Codex 今日、本月、总计 Token 来自当前登录账号官方 `account/usage/read`，统一使用“万 / 亿”单位。
- 按官方接口真实返回的窗口显示周额度，不虚构已经缺失的 5 小时额度。
- Codex 与 Claude Code 数据互不混用；Claude Code 个人账号不显示伪造的 Token 总量。
- 支持 DeepSeek 账户余额与 OpenRouter 当前 Key 剩余额度。

## 界面实现

macOS 使用 AppKit/Core Animation，Windows 使用 WPF/Acrylic。液态玻璃、动态液位、气泡、刻度、扫光、连续连接颈部和进度条全部由代码实时绘制，不使用效果截图贴图。

## 隐私与安全

- 不读取或上传聊天正文、会话日志、浏览器 Cookie 或 Codex `auth.json`。
- 不包含遥测、广告或第三方分析。
- API Key 不写入 Balance Capsule 设置、日志或界面，只在受限的同主机 HTTPS 余额请求期间使用。
- 本地仅保存悬浮球位置、所选数据源、显示状态等非敏感设置。

## 首次运行提示

两个平台的当前发布包均没有商业开发者证书签名。请只从本仓库 Release 下载，并使用随附的 `BalanceCapsule-SHA256SUMS.txt` 校验文件完整性。macOS 首次启动请在 Finder 中右键应用并选择“打开”；Windows 若出现 SmartScreen，请核对来源和哈希后选择“更多信息”。
