# Balance Capsule 1.2.15-win.14

- 仅支持 Windows 11 x64；提供未压缩单文件版和更可靠的多文件便携 ZIP，无需另装 .NET。
- 修复启动早期异常无提示退出：现在显示错误并写入 `%LOCALAPPDATA%\BalanceCapsule\startup.log`。
- CC Switch SQLite 仅在数据库实际存在时初始化，系统 SQLite 不可用时不再阻止主程序启动。
- 78px 代码绘制液态玻璃悬浮球，鼠标悬停自动打开连续液滴轮廓详情面板。
- 详情面板使用 Windows Acrylic 背景模糊、半透明渐变、高光与低强度色散。
- Codex 今日、本月、总计 Token 读取当前登录账号官方 `account/usage/read`，只使用“万 / 亿”单位。
- 按真实窗口时长识别一周额度，不再虚构已经缺失的 5 小时额度。
- 双击悬浮球立即刷新；右键与托盘菜单支持刷新、切换 Codex/Claude Code、显示或隐藏悬浮球。
- Claude Code 个人账号不显示伪造 Token 总量，也不会混用 Codex 数据。
