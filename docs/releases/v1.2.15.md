# Balance Capsule 1.2.15-mac.13

- 原生 Apple Silicon（arm64）版本，最低支持 macOS 13。
- 应用图标改为全画布液态玻璃，由 macOS 只裁切一次外轮廓。
- 78px 常驻悬浮球、液面、气泡、刻度、连接颈部和详情面板均由代码实时绘制，不使用静态设计贴图。
- 鼠标滑入自动展开详情，双击悬浮球立即刷新，拖动时支持屏幕边缘吸附。
- Codex 今日、本月、总计 Token 来自当前登录账号官方 `account/usage/read`；仅使用“万 / 亿”单位。
- 支持 Codex、Claude Code、DeepSeek 与 OpenRouter 数据源，来源之间的数据互不混用。
