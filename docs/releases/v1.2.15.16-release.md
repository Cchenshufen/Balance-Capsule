# Balance Capsule v1.2.15.16

本次修订统一 Windows 与 macOS 的液态玻璃详情面板视觉。

## 下载

- macOS 26+ Apple Silicon：`BalanceCapsule-1.2.15-mac.14-arm64.dmg`
- Windows 11 x64 单文件：`BalanceCapsule-1.2.15-win.16-x64.exe`（直接下载）
- Windows 11 x64 便携包：`BalanceCapsule-1.2.15-win.16-x64.zip`（备用）

## Windows UI 修复

- Acrylic 仅作用于液态连接颈和圆角面板，不再显示透明窗口边界形成的白色矩形。
- 玻璃底色、蓝红色散描边、内层高光和背景模糊强度按 macOS 版重新校准。
- 标题改为 17px 常规字重，Codex 胶囊改为 78px，并使用三层代码绘制标识。
- 错误态不再覆盖粉色卡片，只在玻璃面板中显示错误说明；隐藏无效的额度行。
- 保留 `1.2.15-win.15` 的 Windows 启动绑定修复。

Windows 版仅支持 Windows 11 x64。默认直接运行单文件 EXE；若被安全软件阻止，再使用完整解压后的便携 ZIP。
