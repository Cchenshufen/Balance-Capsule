# Balance Capsule v1.2.15.15

本次修订修复 Windows 版双击后没有任何界面、进程直接退出的问题。用户提供的 `startup.log` 显示详情窗口将只读额度属性按双向方式绑定，WPF 在首次显示窗口时抛出异常；当前版本已改为显式单向绑定，并加入回归测试。

## 下载

- macOS 26+ Apple Silicon：`BalanceCapsule-1.2.15-mac.14-arm64.dmg`
- Windows 11 x64：`BalanceCapsule-1.2.15-win.15-x64.zip`（推荐）
- Windows 11 x64 单文件：`BalanceCapsule-1.2.15-win.15-x64.exe`

## Windows 修复

- 修复 `CurrentRemainingPercent` 与 `WeeklyRemainingPercent` 只读属性被 WPF `ProgressBar.Value` 默认双向绑定引发的启动异常。
- 两个进度条现在显式使用 `OneWay`，详情窗口可以正常显示。
- 加入绑定模式回归测试，防止相同问题再次进入发布包。
- 保留启动日志：若仍有环境相关错误，可查看 `%LOCALAPPDATA%\BalanceCapsule\startup.log`。

Windows 版仅支持 Windows 11 x64。推荐完整解压便携 ZIP 后运行 `BalanceCapsule.exe`。
